# GDSC_Git_Workshop_Demo
Demo Repo for the GDSC Git Workshop

## Task
There is a bug in `calculator.py`.

### Your job/objective
1. Fork this repo
2. Clone your forked repo
3. Create a branch for bug fixing
4. Fix the bug/s
5. Commit your change and
6. Push to your repo
7. Open a pull request to the main repo
